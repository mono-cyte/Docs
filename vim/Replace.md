# Replace Mode

## Switch

|     |              |
| --- | ------------ |
| `R` | Replace Mode |

> [!NOTE]
> `R` replaces the character and moves the cursor forward

## Editing

|                 |                           |
| --------------- | ------------------------- |
| `Backspace`     | undo the last replacement |
| `Delete`        | removes the character     |
| `↑`/`↓`/`←`/`→` | move the cursor           |
