# macro

|              |                                                        |
| ------------ | ------------------------------------------------------ |
| `q[reg]`     | start recording a macro and store it in register [reg] |
| `q`          | stop recording                                         |
| `@[reg]`     | execute macro in register [reg]                        |
| `@@`         | execute last macro                                     |
| `g@[motion]` | call 'operatorfunc'                                    |
