# Visual Mode

## Switch

|          |                           |
| -------- | ------------------------- |
| `v`      | Visual mode               |
| `V`      | Visual line(row) mode     |
| `Ctrl v` | Visual block(column) mode |

## Editing

| `gv`             | reselect the previous Visual area                                                                                                                                              |
| `gV`             | don't reselect the previous Visual area when executing a mapping or menu in Select mode                                                                                        |
